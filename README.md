## URL
root/		        => ログイン画面
root/admin/             => 管理者画面
root/books/             => 本一覧
root/books/create	=> 本の登録画面
root/books/<title>	=> <title>の詳細ページ
root/books/<title>		=> 本<title>詳細
root/books/<title>/update/	=> <title>の更新ページ
root/books/<title>/delete/	=> <title>の削除ページ
root/users/			=> ユーザ一覧
root/users/<name>		=> ユーザ<name>詳細
